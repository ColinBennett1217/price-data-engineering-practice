# Price Data Engineering Practice

This repo contains data engineering practice work to prepare for a data engineer role at a building systems / HVAC company.

## Datasets

- **US EIA Manufacturing Energy Consumption Survey (MECS)**  
  - 2022 data tables: https://www.eia.gov/consumption/manufacturing/data/2022/

- **Singapore Building Energy Performance**  
  - Listing of Building Energy Performance Data: https://data.gov.sg/datasets?search=building%20energy%20performance

- **Building Energy Usage (Kaggle)**  
  - https://www.kaggle.com/datasets/programmer3/building-energy-usage-dataset

More datasets will be added as the project evolves.

## Goals

- Practice building data pipelines (Excel/CSV → cleaned tables → analysis).
- Explore energy consumption by industry, building type, and region.
- Document data quality issues and root-cause style investigations.
